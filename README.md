# Weather Dashboard
## Description
This project is a Python-based Weather Dashboard developed as part of the M602 Computer Programming module. The application integrates with the OpenWeatherMap API to retrieve real-time and forecast weather data for user-specified locations.
It provides an interactive graphical user interface built with Tkinter and visualizes weather data using charts for better understanding and analysis.

## Features
- Real-time weather information (temperature, humidity, wind speed, pressure)
- 5-day weather forecast
- Data visualization using charts (Matplotlib)
- Favourite cities management using JSON file storage
- Export weather data to CSV
- Responsive and user-friendly GUI
- Exception handling for API and network errors

## How to Run the Project
1. Install dependencies:
pip install -r requirements.txt

2. Run the application:
python Weather_Dashboard.py

## Technologies Used
- Python 3
- Tkinter (GUI development)
- Matplotlib (data visualization)
- OpenWeatherMap API
- JSON & CSV (file handling)
